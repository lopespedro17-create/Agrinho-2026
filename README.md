# 🌱 Agro Forte - Futuro Sustentável
## Projeto para Concurso Agrinho 2026

### 📋 Descrição do Projeto

**Agro Forte** é um website moderno e interativo desenvolvido para o Concurso Agrinho 2026, com o tema: **"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**.

O site apresenta soluções inovadoras que demonstram como a tecnologia, a agricultura e a preservação ambiental podem trabalhar juntas para construir um futuro mais sustentável e próspero.

---

### 🎯 Objetivo

Mostrar de forma criativa e interativa como:
- A tecnologia pode resolver problemas ambientais da agricultura
- Soluções sustentáveis aumentam a produtividade
- A preservação do meio ambiente é viável e rentável
- Jovens podem contribuir para um futuro mais verde

### 📱 Seções do Site

#### 1. **Tela Inicial (Hero)**
- Título impactante: "Agro Forte, Futuro Sustentável"
- Animações flutuantes com emojis temáticos
- Botão explorador com efeito hover
- Design minimalista mas poderoso

#### 2. **O Desafio Atual**
- 4 cards informativos sobre problemas ambientais
- Dados reais: desperdício de água (70%), emissões (24%), desmatamento (80%), biodiversidade (40%)
- Barras de progresso animadas
- Transições suaves ao scroll

#### 3. **Soluções Sustentáveis**
- 6 cards com soluções inovadoras:
  - 💧 Irrigação Inteligente (-60% água)
  - ☀️ Energia Limpa (-45% CO₂)
  - 🚁 Drones e IA (-50% químicos)
  - 🌳 Agrofloresta (biodiversidade)
  - 📊 Agricultura de Precisão (-35% impacto)
  - ♻️ Ciclo Circular (zero desperdício)
- Hover effects com elevação
- Listas de benefícios

#### 4. **Quiz Interativo**
- 5 perguntas sobre agricultura sustentável
- Sistema de pontuação (0-100)
- Feedback visual de respostas corretas/incorretas
- Barra de progresso
- Resultado final com mensagem personalizada
- Opção de refazer

#### 5. **Impacto Mensurável**
- 4 estatísticas com contadores animados
- Círculos de progresso com SVG
- Casos de sucesso globais (Brasil, Europa, Ásia)
- Dados comprovados de práticas sustentáveis

#### 6. **Galeria Visual**
- 6 itens visuais categorizados
- Ícones emoji representando tecnologias
- Gradientes modernos
- Efeito hover com zoom
- Transições suaves

#### 7. **Seção Inspiradora Final**
- Mensagem poderosa sobre o futuro
- Chamada para ação
- Dois botões de CTA
- Background gradiente motivador

#### 8. **Footer**
- Informações do projeto
- Links de navegação
- Seção de contato
- Redes sociais
- Copyright

### 🎮 Funcionalidades Interativas

#### Quiz
- **Clique nas opções** para responder
- **Veja o feedback** imediato (verde = correto, vermelho = incorreto)
- **Acumule pontos**: 20 pontos por resposta correta
- **Receba resultado personalizado** ao final

#### Modo Escuro
- **Clique na lua (🌙)** no header para ativar/desativar
- **Preferência salva** no navegador

#### Navegação
- **Menu responsivo** em dispositivos móveis
- **Links suaves** com scroll automático
- **Sticky navbar** para fácil acesso

#### Animações
- **Barras de progresso** que preenchem ao scroll
- **Contadores** que animam quando vistos
- **Cards** que aparecem com fade-in
- **Hover effects** em todos os elementos interativos
